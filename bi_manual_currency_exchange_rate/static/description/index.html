Manual Currency Exchange Rate

Need to provide invoices, sale order, purchase order to customers from different countries along with currency rates according to their country?
This module is designed to provide currency rate exchange. All you need to do is follow steps shown below.

How it works?

While creating any sale order or purchase order or invoice, you can add currency exchange option to apply manual currency rate to the products. Just allow manual currency exchange rate and apply rates according to currenct.


On create invoice page there is a check-box that you need to select for manual currency exchange rate. Apply rates that you want to be exchanged.


While creating sale order, there is a check-box that you need to select for manual currency exchange rate. Apply rates that you want to be exchanged.


While creating purchase order there is a check-box that you need to select for manual currency exchange rate. And apply rates that you want to be exchanged.


Following steps remains same for Sale order and Pruchase order


Provide the rates you want to be converted in invoice in input field that will appear after you check Apply Manual Currency Rate.


Once you have applied manual currency exchange rates, add any product to invoice. Price of the product will changed according to the rates.
